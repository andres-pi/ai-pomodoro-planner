# AI Pomodoro Planner

A Pomodoro timer designed to improve focus and productivity.  
It includes a traditional Pomodoro mode and an AI-powered planning mode that generates personalized study schedules based on user goals and deadlines.

---

# What this system does

This application helps users organize their focus sessions using the Pomodoro technique.

The system supports two modes:

### Normal Mode

Allows manual configuration of Pomodoro sessions:

- Base focus time
- Break time between sessions
- Number of sessions
- Long break after finishing sessions
- Custom colors for the timer interface

This mode behaves like a traditional Pomodoro timer but with extended configuration.

### Custom Mode (AI Assisted)

The custom mode allows users to create a study plan automatically.

Users can provide:

- A target date (for example an exam date)
- Number of study days per week
- Number of topics / modules to study

The AI processes this information and generates an optimized Pomodoro schedule designed to reach the goal before the deadline.

The generated plan is then shown in the Pomodoro configuration screen where the user can:

- Accept the suggested configuration
- Modify the sessions
- Adjust durations

This mode can also send reminders and notifications for scheduled study sessions.
